Search Bars based on Platforms


A search box, search field or search bar is a graphical control element used in computer programs,
such as file managers or web browsers, and on web sites. A search box is usually a single-line text box
or search icon ( which will transform into a search box on click activity) with the dedicated function
of accepting user input to be searched for in a database.
